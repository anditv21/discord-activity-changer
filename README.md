<h1 align="center">Welcome to discord-activity-changer 👋</h1>


> A simple discord activity changer in the form of a selfbot.

## Requirements

- discord~=1.7.3


## Install

```sh
pip install -r requirements.txt
```

## Start command

```sh
python main.py
```

## Example Usage in Discord

```sh
>>>rpc w TV
```

## Activity Types
* w = watching
* p = playing
* l = listening
* s = streaming
